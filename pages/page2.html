What is my solution?

Another mentor and coach will inform you that an idea is worthless (just like life?). My pyramid keyboard initially seemed such a simple and common solution that it seemed like it had always been there (and this turned out to be untrue). Then the Keyboard Luddites declared that no one needed it.

Now that it turns out that only such keyboard makes Touch Typing easy as using a mouse, the reviews have become much warmer. The combination AR glasses and my keyboard built into a mouse is suitable for any work with software.

But data labeling for AI solves a very pressing problem here. I'm talking about machine vision. The way a car doesn't drive blind is the result of the painstaking work of thousands of people called data labeling. This is primitive and unproductive work. Because they also work as if "blind", using photographs. The ability to directly see an object on the road and immediately mark it in augmented reality will provide the neural network with very useful information, which can be called an "Impression". By conveying your emotions about the road, you convey an image. This is something that huge amounts of data cannot replace.
